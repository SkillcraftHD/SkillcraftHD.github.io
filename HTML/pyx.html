New Project

A new big project I'm working on, my first own game engine called Pyx! Programmed in C++ and still in early development, I'm focusing mainly on lightweightness, simplicity and stability.

Uses the GLFW graphics library for a balance between low-level customability and simplicity while focusing on 2D graphics.

I plan to release it completely for free, maybe even open-source. No account needed, just hop right in!

It will most likely only be available to Windows users for the sense of immense extra work.
